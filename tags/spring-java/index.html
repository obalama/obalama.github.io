Spring 中Aware接口的理解

从Spring中POJO类的理解来讲，注入到容器中的Bean一般是不用太理会自己所处环境的信息，不用理会是谁创建了自身，不用理会自己在所处的环境中标识是什么，也不用在意加载自己的classLoader是哪位。但是如果知道了这些信息，对于我们的调试时相当有帮助的。那么，怎么实现呢 ？
